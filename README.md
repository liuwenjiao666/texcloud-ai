# TexCloud AI · 客服工作台（更换技能版 · 中英双语）

客服经理工作台 HTML 高保真原型，演示「待办驱动 + 技能工具箱 + 宽画布分析」的双模式交互，支持中英文一键切换（智能助手中文名「小e」，英文名「Eon」）。

当前版本技能工具箱采用洗涤助手 10 项技能（含 2 项尚未开放）：点击技能进入宽对话画布，展示技能使用引导与推荐问题；点击推荐问题会填入输入框，由用户确认后发送。

## 在线预览

开启 GitHub Pages 后访问：

**https://liuwenjiao666.github.io/texcloud-ai/**

## 本地打开

直接用浏览器打开 `index.html` 即可，无需安装依赖。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | GitHub Pages 入口（更换技能版 · 中英双语，内容与下方源文件一致） |
| `客服工作台-更换技能版.html` | 当前主版本源文件，便于本地识别文件名 |
| `客服经理 · 技能工具箱（10 项，含 2 项尚未开放）.md` | 技能工具箱内容与交互说明文档 |
| `客服经理工作台-原型-双模式-双语.html` | 历史版本：双模式 · 中英双语 |
| `客服经理工作台-原型-双模式.html` | 历史版本：仅中文的双模式 |

> 右上角 `中 / EN` 按钮可一键切换语言。

## 开启 GitHub Pages

1. 仓库 **Settings** → **Pages**
2. **Source** 选 `Deploy from a branch`
3. **Branch** 选 `main` / `/ (root)`
4. 保存后等待 1～2 分钟，访问上方链接
