# TexCloud AI · 客服经理工作台（方案二 · 双模式）

客服经理工作台 HTML 高保真原型，演示待办驱动 + 技能工具箱 + 宽画布分析的双模式交互。

## 在线预览

开启 GitHub Pages 后访问：

**https://liuwenjiao666.github.io/texcloud-ai/**

## 本地打开

直接用浏览器打开 `index.html` 即可，无需安装依赖。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | GitHub Pages 入口（双模式方案） |
| `客服经理工作台-原型-双模式.html` | 同上，便于本地识别文件名 |

## 开启 GitHub Pages

1. 仓库 **Settings** → **Pages**
2. **Source** 选 `Deploy from a branch`
3. **Branch** 选 `main` / `/ (root)`
4. 保存后等待 1～2 分钟，访问上方链接
